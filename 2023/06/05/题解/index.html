大致题意

有n名员工（1号是老板自己） k次讨论 每次讨论员工都有自己的意见 在讨论前老板可以 让一个人直接滚蛋（老板不能自己t自己）

对于一次讨论 人多的留下 平局的话都退出

找最后剩下员工的最大数

思路

不管怎么讨论 老板肯定是不能走的 因此老板给出的意见才是正确意见 直接找有几个 员工和老板一个意见就行了

大致题意

给一个n 构造一个数列a (1 <= ai <= n) 不重复 要求构造的数列中 任何子数列的和都不能被子数列的长度整除 也就是区间l - r 的数的和 不能被 r - l + 1 整除 （1 <= l < r <= n）

思路

首先 初始数组 肯定是 1 2 3 .. n

n = 1的时候 数列中只有 1 个数 显而易见 答案就是1

然后讨论 n 为奇数的时候 对于最大的区间 1 - n 和为 （1 + n） * n / 2 可以被n整除 所以无论怎么构造都不会成功

n 为 偶数的时候 分为两种情况

l 和 r 有不同的奇偶性 他的和 为 （r - l + 1)(l + r) / 2 不可以被 r - l + 1 整除 满足题意

l 和 r 有相同的奇偶性 和为 （r - l + 1)(l + r) / 2 可以被 r - l + 1 整除 不满足题意 但是 sum - 1 或 sum + 1满足题意 因为r - l + 1 >= 2 ,因此我们需要让区间最少3内的和 + 1 或者 -1 容易想到 只需要交换相邻的数 就可以构造出满足题意的数组

B

大致题意

给一个长度为n的数列 执行0或多次的交换操作 或者 执行一次 预前交换 在执行普通的交换操作

ps： 普通操作的交换有条件的：只能对|i - j| = k 的两个位置交换

如果 是第一种 能排好序的话 就输出0 第二种的话 输出1

如果无论如何都不能的话 输出-1

思路

普通操作就相当于 每次移动固定的 m 个长度 如果 当前数的位置 到 排好序的位置 % m != 0 的话 就无法只通过普通排序来完成

预前交换 可以随意交换两个数 所以 如果当前的数列通过普通排序 排序后，如果有问题肯定会有至少两个位置的数 不在原本的位置 这时候可以通过预前交换来解救 但是只能执行一次

所以只需要统计普通排序后 有多少数 回不到原本的位置即可

大致题意

在坐标图上跳来跳去 起点是（0，0） 终点给出 要求在跳两下 到达终点 并且 跳跃时 跳的起点和 跳的下一步之间的连线不能经过整数坐标

思路

两个点 (x1,y1) 和 （x2,y2）如果 |x1 - x2| = 1 或者 |y1 - y2 | = 1 的话 就是一个边长1 ，x的矩形 两个点之间没有x的整数（不考虑端点） 所以直接走到 （n - 1 , 1）然后直接走到 终点即可

B

大致题意

给一个n 构造一个 2 * n 的网格 从起点（1，1）开始走 只能向下或者右 请你给网格中的点赋值 使最终的路径的最小代价最大化

一条路径的代价是写入路径中单元格上的数字的交替和。也就是说，让写在单元格上的数字是a1 a2…ak
(按照访问的顺序)，路径的代价为a1−a2+a3−a4+…=∑ki=1ai·(−1)i+1

思路

容易想到 起点和终点都是必过点 不妨让起点的价值最大 终点的价值次大 然后考虑其他点 我们肯定是希望访问次数多的点 价值尽可能大 对于（i + j） 是奇数的点 （i 和 j 从0开始） 都是要减去的值 对于相加是偶数的点 都要加 显然，所有要添加的值都严格大于所有要减去的值是最优的
