本地部署

一般来说按照大佬的readme文档就可以了，下面将分享我的部署经历

  1. 创建虚拟环境
    python -m venv venv

  2. 激活虚拟环境
    venv\Scripts\activate

    此处和大佬的readme文档不同

  3. 安装要求的python组件
    pip install -r requirements.txt

  4. 参数配置修改

    1. 在 config.ini 添加你的api key
    2. 修改./readconfig/mycofig.py 的base 绝对路径 使其为config.ini的文件夹路径
    3. 将config.ini中的api链接改为openai-proxy
    4. 这里也改

  5. 启动程序
    python application.py

  6. 发现部分包还未添加，如pptx,markdown,openai等

    缺什么加什么就行，pip install xxx

  7. 下载Redis，开启

  8. 重新运行即可
