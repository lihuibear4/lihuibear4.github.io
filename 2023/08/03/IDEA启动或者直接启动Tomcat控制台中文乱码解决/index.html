中文乱码原因

由于安装的windows系统本地语言选择的是中文，也就是GBK编码；而IDEA或者Tomcat日志使用的是UTF-8编码，这就导致了中文显示乱码。

启动Tomcat中文乱码解决

  • 找到Tomcat日志配置文件 conf\logging.properties

IDEA其他配置一律UTF-8编码配置

File->Settings 搜索 File Encodings ,编码统一UTF-8

改为GBK编码是解决控制台显示中文乱码问题，其他UTF-8编码是为了与编辑器、浏览器等保持一致通用
